# OrbitPal
AR Solar System Explorer — Unity 6, AR Foundation, Android

## Requirements
- Unity 6.x
- AR Foundation 6.x  
- Google ARCore XR Plugin
- DOTween (dari Asset Store)
- Planets of the Solar System 3D (dari Asset Store)

## Setup
1. Clone repo
2. Buka di Unity 6
3. Install packages di Package Manager
4. Import asset planet dari Unity Asset Store
5. Build ke Android
